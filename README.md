# weapp_github
github微信小程序，wepy框架编写

## 使用步骤
> 小程序开发者工具打开时，直接打开项目目录即可，打开不校验合法域名、web-view(业务域名)、TLS版本以及HTTPS证书开关，关闭ES6转ES5、上传代码时样式自动补全等选项

1、`git clone https://github.com/shenyiling/weapp_github.git`</br>
2、`cd weapp_github && npm i`</br>
3、`wepy build --watch`
